# AgentManager (Ratatui TUI)

This branch resets the project into a clean terminal UI baseline using [Ratatui](https://ratatui.rs/).

## Run

```bash
cargo run
```

## Controls

- `j` / `Down`: next worktree
- `k` / `Up`: previous worktree
- `q`: quit

## Notes

- This is intentionally minimal to support rebuilding the product as a TUI.
- Old GPUI code and prior workspace crates were removed on this branch.
